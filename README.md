# Mobile Development with React Native - ProDev Setup

This repository contains a series of tasks focused on modern mobile development using **React Native**, **TypeScript**, and the **Expo Framework**. The project covers environment setup, core component implementation, responsive styling, and UI construction.

## Project Timeline
- **Start Date**: Jan 5, 2026
- **End Date**: Jan 12, 2026
- **Status**: Completed

## Technologies Used
- **React Native**: Cross-platform mobile framework.
- **TypeScript**: Static typing for enhanced development productivity.
- **Expo & Expo Go**: Framework for rapid development and physical device testing.
- **Expo Router**: File-based routing for React Native.
- **Vector Icons**: Ionicons and FontAwesome for UI elements.

## Directory Overview

### 0. [mobile-development-setup](./mobile-development-setup/)
- **Objective**: Environment configuration.
- **Key Features**: Documentation of Node.js, WebStorm, and Expo Go setup on physical hardware.

### 1. [prodev-mobile-app-0x00](./prodev-mobile-app-0x00/)
- **Objective**: Initial Scaffolding.
- **Key Features**: Understanding the Expo project structure and the `reset-project` workflow.

### 2. [prodev-mobile-app-0x01](./prodev-mobile-app-0x01/)
- **Objective**: Core Components and StyleSheet.
- **Key Features**: Implementation of `View` and `Text` components with custom `StyleSheet` objects including `fontVariant` and `textAlign` properties.

### 3. [prodev-mobile-app-0x02](./prodev-mobile-app-0x02/)
- **Objective**: Layout and Media.
- **Key Features**: responsive layout using `SafeAreaView`, `ImageBackground`, and `Dimensions` to create a high-fidelity landing screen.

### 4. [prodev-mobile-app-0x03](./prodev-mobile-app-0x03/)
- **Objective**: Advanced UI Components.
- **Key Features**: Creation of a functional Sign-In screen utilizing `TextInput`, `TouchableOpacity`, and social login integrations.

## How to Run
To run any of the applications:
1. Navigate to the task directory: `cd prodev-mobile-app-0x0X`
2. Install dependencies: `npm install`
3. Start the server: `npx expo start`
4. Scan the QR code with the **Expo Go** app on your physical device.
